Fetch Demo

  • It is a method! i.e a function.
  • Used to get or push some data from/to an api endpoint.
  • Returns a Promise. (?!)
  • fetch(url, options)
  • Methods:
    • GET (Pull data from api endpoint)
    • POST (Create a new object and push it to api endpoint)
    • PUT (Update an existing object)
    • DELETE (Delete an existing object)
